## Run Locally

Clone the project

```bash
  git clone https://github.com/ozgeahras/meme-generator-with-react.git

```

Go to the project directory

```bash
  cd ..meme-generator-with-react
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run dev
```
